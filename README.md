# Netflix-Churn-Analysis
Netflix Churn Analysis - EDA Exploratory Data Analysis on Netflix user data to identify patterns and key factors influencing customer churn using visualizations and statistical insights.


## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on Netflix user data to understand customer churn behavior. The goal is to identify patterns and key factors that influence whether a customer stays or leaves the service.

## 🎯 Objectives
- Analyze customer data to detect churn patterns
- Identify important features affecting user retention
- Visualize trends using graphs and charts
- Provide insights for improving customer retention strategies

## 📂 Dataset
The dataset includes information such as:
- User demographics
- Subscription details
- Account activity
- Churn status

*https://www.kaggle.com/datasets/abdulwadood11220/netflix-customer-churn-dataset*

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🔍 Exploratory Data Analysis
The analysis includes:
- Data cleaning and preprocessing
- Handling missing values
- Univariate analysis
- Bivariate analysis
- Correlation analysis
- Visualization of churn distribution

## 📊 Key Insights
- Identified factors contributing to customer churn
- Observed trends in subscription types and usage
- Highlighted potential areas to improve retention
